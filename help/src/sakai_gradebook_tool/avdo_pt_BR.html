Importar notas de itens existentes no Quadro de Notas

O Quadro de Notas permite importar notas para os itens existentes no Quadro de Notas, bem como Itens que ainda não foram criados. Para importar notas e criar um item Quadro de Notas novo, veja Importar notas de uma planilha. Para importar notas de itens existentes no Quadro de Notas:

  1. No menubar do seu site, clique em Quadro de Notas.

  2. Perto do topo, clique em Importar Notas.

  3. Baixar o modelo de planilha clicando em Baixar Modelo de Planilha.

  4. Editar a planilha em sua aplicação, como o Excel, e salvar como arquivo .csv.

  5. Importar a planilha clicando em Procurar... e escolhendo o arquivo .csv .

  6. Depois de selecionar o arquivo, clique em Importar Planilha.

  7. A página seguinte deve mostrar as notas importadas. Quando terminar, clique em OK.

Considerações

  • Você não pode importar notas para os itens Quadro de Notas enviados a partir da ferramenta Atividades.

  • Notas importadas usando a opção Importar Notas substituirá todas as classes pré-existentes. Use sempre o modelo de planilha baixado no passo 3.
