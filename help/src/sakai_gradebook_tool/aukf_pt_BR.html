Importar notas da planilha

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Resumo.

Em Quadro de Notas, você pode importar notas de duas maneiras diferentes:

  • Importar uma planilha inteira. Este método substitui quaisquer notas existentes no Quadro de Notas para colunas com o mesmo nome como itens quadro de notas existentes, criando novos Itens do quadro de notas para quaisquer colunas que não correspondem a Itens do quadro de notas existentes.

  • Importar itens individuais do quadro de notas. Este método cria novos itens do quadro de notas para cada coluna da planilha de importar. Você não pode importar todas as colunas com o mesmo nome como itens quadro de notas existentes.

Você também pode usar a ferramenta Post'Em fazer upload de uma planilha de notas .csv (e feedback, se desejado) que alunos podem ver; veja Usar a ferramenta Post'Em.

Observação: Você não pode importar notas para os itens quadro de notas adicionados a partir da ferramenta Atividades; veja Vincular uma atividade ao Quadro de Notas.

Importar uma planilha inteira

  1. No menubar do seu site, clique em Quadro de Notas.

  2. Perto do topo, clique em Importar Notas.

  3. Baixe o modelo de planilha, clicando em Baixar Modelo de Planilha, escolha Salvar Arquivo, e clique em OK.

  4. Editar a planilha em um aplicativo de planilha, como o Excel, e salve-o como um arquivo .csv.

  5. Importar a planilha clicando em Procurar... e escolhendo o arquivo .csv file.

  6. Depois de selecionar o arquivo, clique em Importar Planilha.

  7. A página seguinte deve mostrar as notas importadas. Quando terminar, clique em OK.

Ao usar esta opção, é melhor fazer o download do modelo de planilha fornecida, conforme indicado no passo 3 acima. Você pode enviar uma planilha diferente no formato .csv, mas se o fizer, formatar o mais próximo possível do modelo. A primeira coluna deve conter nomes individuais, e a primeira linha deve conter títulos, com o título e valor de ponto para cada item do quadro de notas como neste exemplo:

Teste 1 [100.0]

Importar item individual do Quadro de Notas

Para importar Itens do quadro de notas, primeiro crie um arquivo de planilha . csv contendo colunas de ponto com base em pontuações, e, em seguida, enviá-la para o carregamento do Quadro de Notas. A primeira linha de sua planilha deve conter cabeçalhos para as colunas que ele contém. A primeira coluna deve conter nomes de usuários individuais.

Depois de ter seu arquivo .csv pronto para upload:

Enviando um arquivo para plataformas de carga

  1. No menubar do seu site, clique em Quadro de Notas.

  2. Perto do topo, clique em Importar item do Quadro de Notas da planilha.

  3. Na tela de Importar item do Quadro de Notas, clique em Carregar planilha (formato csv) para plataformas de carregamento.

  4. No campo ao lado de "Título", digite um nome exclusivo para ser exibido na plataformas de carregamento.

  5. Para selecionar um arquivo para upload, clique em Procurar. Localize o arquivo no seu computador, clique em seu nome, e clique emAbrir.

  6. Clique em Salvar.

  7. Na tela de carregamento, rever o número de itens e alunos para o upload, e ver uma exibição de conteúdo do arquivo. Se o conteúdo está correto, clique em Ok. Se você precisa selecionar outro arquivo, clique emVoltar.

Uma vez que um arquivo é enviado para plataformas de carregamento, os seus conteúdos se tornam disponíveis para importar como um item novo quadro de notas. Para importar conteúdo de um arquivo enviado como um item novo quadro de notas:

Importar uma coluna de um arquivo no Carregando

  1. Na tela Importar item do Quadro de Notas, encontrar o seu arquivo recém-carregado na plataforma de carregamento, e ao lado dele, clique em Importar.

  2. Na tela de visualização de Importação, clique no botão para selecionar a coluna que representa o item quadro de notas e pontuações correspondentes que deseja importar. (Você pode selecionar apenas um de cada vez.) Em seguida, clique Importar Selecionado.

  3. Na tela Importar item do Quadro de Notas e Pontuações, insira a seguinte informação:

    • Título: O título que você forneceu nas etapas anteriores é automaticamente preenchido. Se você não fornecer um título antes, você precisa fornecer um aqui. O comprimento máximo é de 255 caracteres.

    • Valor do Item do Quadro de Notas: Digite um valor total de pontos para o item quadro de notas. Este deve ser maior do que zero. Valores decimais são permitidos até duas casas decimais.

    • Data de Conclusão: No formato mm/dd/yy, indica a data de conclusão do item, ou clique no ícone de calendário ( 
       ) para selecionar uma data usando a ferramenta de calendário. Este campo não é obrigatório.

    • Selecione uma coluna para importar comentários: Se uma coluna no seu arquivo contém comentários, use a lista para selecionar a coluna apropriada e adicionar os comentários para a importação. Comentários aparecerão ao lado das notas do Quadro de Notas. Este campo não é obrigatório.

  4. Para marcar este item como disponível para os alunos, selecione Liberar esse item para os Alunos. Você pode desmarcar mais tarde, se necessário.

  5. Para incluir os este item no cálculo da nota do curso, selecione Incluir esse item no cálculo da nota do curso.Você pode desmarcar mais tarde, se necessário.

  6. Quando finalizar, clique em Submeter. Você retornará para a tela de Visualizar Importação, onde você verá a confirmação de que o item foi adicionado. Agora você pode selecionar outro item para a importação, ou clique no ícone ( 
     ) para retornar ao menu principal; veja O botão de resetar. Você deverá ver a sua atividade importada listada em "Resumo dos Itens do Quadro de Notas".
