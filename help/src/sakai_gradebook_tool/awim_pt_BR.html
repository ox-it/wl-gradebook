Imprimir a partir do Quadro de Notas

Embora não há atualmente nenhuma função de impressão na ferramenta Quadro de Notas, você pode usar recursos de seu navagador para imprimir a partir do Quadro de Notas. Estes métodos imprimirão várias páginas.

Para instrutores

Você pode imprimir uma lista que mostra todas as notas dos alunos para um item do quadro de notas ou uma lista mostrando todos os itens do quadro de notas para um aluno. Para imprimir outras listas, você deve usar o Exportar Quadro de Notas ou Exportar Notas do Curso. Para mais informações, veja Usar o Quadro de Notas para ver e exportar notas dos aluno.

Impressão de uma lista que mostra todas as notas dos alunos para um item do quadro de notas

  1. A partir da página Itens do Quadro de Notas, passe mouse sobre o item do quadro de notas você deseja imprimir.

  2. Botão direito do mouse (Ctrl-clique no Mac) e siga as instruções apropriadas abaixo para seu navegador:

    • Firefox: No menu que aparece, escolha Abrir Link em uma Nova Janela ou Abrir Link em uma Nova Aba. A nova aba ou janela irá exibir a página Itens do Quadro de Notas. Clique no nome da atividade para exibir as notas dos alunos.

    • Internet Explorer 7 ou superior: No menu que aparece, escolha Abrir Link em uma Nova Janela ou Abrir Link em uma Nova Aba. A nova aba ou janela irá exibir a página Itens do Quadro de Notas.

  3. Se você tiver mais que 50 alunos, você pode selecionar Ver todos a partir da lista à direita, acima dos nomes dos alunos.

  4. No menu Arquivo, escolha Imprimir... para imprimir a página.

Imprimir uma lista mostrando todos os itens do quadro de notas para um aluno

  1. Na página Todas as Notas, passe mouse sobre o nome de um aluno.

  2. Botão direito do mouse (Ctrl-clique no Mac) e siga as instruções apropriadas abaixo para seu navegador:

    • Firefox: No menu que aparece, escolha Abrir Link em uma Nova Janela ou Abrir Link em uma Nova Aba. A nova aba ou janela irá exibir a página Itens do Quadro de Notas.
    • Internet Explorer 7 ou superior: No menu que aparece, escolha Abrir Link em uma Nova Janela ou Abrir Link em uma Nova Aba. A nova aba ou janela irá exibir a página Itens do Quadro de Notas.

  3. No menu Arquivo, escolha Imprimir... para imprimir a página.

Para alunos

Para imprimir uma lista de todas as suas notas em um curso:

  1. No menubar, escolha Quadro de Notas.

  2. Botão direito do mouse (Ctrl-clique no Mac) e siga as instruções apropriadas abaixo para seu navegador:

    • Firefox: No menu que aparece, do mouse Este Frame, e escolha Abrir Frame em uma Nova Janela ou Abrir Frame em uma Nova Aba. A nova aba ou janela irá exibir a página Itens do Quadro de Notas.
    • Internet Explorer 7 ou superior: No menu que aparece, escolha Abrir Link em uma Nova Janela ou Abrir Link em uma Nova Aba. A nova aba ou janela irá exibir a página Itens do Quadro de Notas.

  3. No menu Arquivo, escolha Imprimir... para imprimir a página.
