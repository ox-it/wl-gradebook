Visualizar notas do curso

Em Quadro de Notas, alunos e instrutores podem visualizar as notas atuais.

Qlunos

Quando você entra Quadro de Notas, dependendo das informações que seu instrutor tornou disponível, você verá uma ou ambas das seguintes opções:

  • Nota de Curso: A nota do curso é uma nota cumulativa com base nos itens que você concluiu. Qualquer item que você não concluiu, não será incluído no cálculo nota do curso, a menos que seu instrutor tenha expressamente atribuído zeros a esses itens. Uma porcentagem da nota aparece entre parênteses e mostra a porcentagem real derivada de itens concluídos.

  • Itens do Quadro de Notas: A lista de itens do quadro de notas mostra o nome de cada item, sua data de conclusão (se aplicável), sua nota, e todos os comentários associados com a nota.

Instrutores

Definir opções de visualização de notas

Os instrutores podem optar por disponibilizar Itens individuais do quadro de notas, bem como notas cumulativas do curso. Para mais informações, veja Tornar notas disponíveis para os alunos.

Visualizar notas

Quando você entrar no Quadro de Notas, você verá uma média do curso para a classe e uma lista de itens individuais do quadro de notas. Você pode visualizar as notas cumulativas utilizando uma das seguintes opções:

  • Perto do topo, clique em Notas do curso. Você verá uma lista de seus alunos. Para a maioria dos quadros de nota, os pontos ganhos, a nota de letra e uma porcentagem é exibida para cada aluno. (A exibição pode variar de acordo com sua configuração no quadro de notas.) A nota de letra e porcentagem baseiam-se nos itens que foram concluídos. Para incluir os itens que não foram incluídos no cálculo da nota do curso, você deve digitar um zero (0) para cada item.

    Para visualizar a nota do curso por seção ou grupo, use o "Visualizar" da lista para selecionar a seção ou grupo. Além disso, você pode usar a caixa de texto para pesquisar para cada aluno.

    Por padrão, apenas 50 alunos são visíveis. Use o "Exibir" da lista para exibir mais alunos na lista.

  • Perto do topo, clique em Todas as Notas. Você verá uma lista de seus alunos, e uma nota do curso e um valor de ponto para cada item do Quadro de Notas. A nota do curso só é mostrada como uma porcentagem e baseia-se nos itens que foram concluídos. Para incluir os itens que não foram incluídos no cálculo da nota do curso, você deve digitar um zero (0) para cada item.

    ara visualizar a nota do curso por seção ou grupo, use o "Visualizar" da lista para selecionar a seção ou grupo. Além disso, você pode usar a caixa de texto para pesquisar para cada aluno.

    Por padrão, apenas 50 alunos são visíveis. Use o "Exibir" da lista para exibir mais alunos na lista.
